Для заданного rootдвоичного дерева вернуть порядок обхода значений его узлов (т.е. слева направо, уровень за уровнем).

Пример 1:

Вход: корень = [3,9,20,null,null,15,7]
Выход: [[3],[9,20],[15,7]]
Пример 2:

Вход: корень = [1]
Выход: [[1]]
Пример 3:

Вход: корень = []
Выход: []

Ограничения:

Число узлов в дереве находится в диапазоне [0, 2000].
-1000 <= Node.val <= 1000
