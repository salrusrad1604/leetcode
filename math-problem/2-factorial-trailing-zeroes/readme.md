Для заданного целого числа nвернуть количество конечных нулей в числеn! .

Обратите внимание, что n! = n * (n - 1) * (n - 2) * ... * 3 * 2 * 1.

 

Пример 1:

Вход: n = 3
 Выход: 0
 Пояснение: 3! = 6, без конечного нуля.
Пример 2:

Вход: n = 5
 Выход: 1
 Пояснение: 5! = 120, один конечный ноль.
Пример 3:

Вход: n = 0
 Выход: 0
 

Ограничения:

0 <= n <= 104
