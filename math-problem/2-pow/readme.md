Реализуйте pow(x, n) , которая вычисляет xвозведенное в степень число n(т.е. ).xn

 

Пример 1:

Вход: x = 2,00000, n = 10
 Выход: 1024,00000
Пример 2:

Вход: x = 2,10000, n = 3
 Выход: 9,26100
Пример 3:

Вход: x = 2,00000, n = -2
 Выход: 0,25000
 Объяснение: 2 -2 = 1/2 2 = 1/4 = 0,25
 

Ограничения:

-100.0 < x < 100.0
-231 <= n <= 231-1
nявляется целым числом.
Либо xне равно нулю, либо n > 0.
-104 <= xn <= 104
