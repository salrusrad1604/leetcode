Для двух строк str1и str2вернуть наибольшую строку x, которая xделит str1иstr2 .

Пример 1:

Ввод: str1 = "ABCABC", str2 = "ABC"
Вывод: "ABC"
Пример 2:

Вход: str1 = "ABABAB", str2 = "ABAB"
Выход: "AB"
Пример 3:

Вход: str1 = "LEET", str2 = "CODE"
Выход: ""
