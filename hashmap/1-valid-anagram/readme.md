Даны две строки sи t, вернуть, trueесли tэтоанаграммаиз s, и falseв противном случае.

 

Пример 1:

Ввод: s = "анаграмма", t = "нагарам"

Вывод: истина

Пример 2:

Ввод: s = "крыса", t = "автомобиль"

Вывод: ложь

 

Ограничения:

1 <= s.length, t.length <= 5 * 104
sи tсостоят из строчных английских букв.
