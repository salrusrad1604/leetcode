Дано rootдвоичное дерево, инвертировать дерево и вернуть его корень .

 

Пример 1:


Вход: корень = [4,2,7,1,3,6,9]
 Выход: [4,7,2,9,6,3,1]
Пример 2:


Вход: корень = [2,1,3]
 Выход: [2,3,1]
Пример 3:

Вход: корень = []
 Выход: []
 

Ограничения:

Число узлов в дереве находится в диапазоне [0, 100].
-100 <= Node.val <= 100
