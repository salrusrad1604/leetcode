Учитывая headсвязный список, поверните список вправо на несколько kпозиций.

 

Пример 1:


Вход: голова = [1,2,3,4,5], k = 2
 Выход: [4,5,1,2,3]
Пример 2:


Вход: голова = [0,1,2], k = 4
 Выход: [2,0,1]
 

Ограничения:

Количество узлов в списке находится в диапазоне [0, 500].
-100 <= Node.val <= 100
0 <= k <= 2 * 109
